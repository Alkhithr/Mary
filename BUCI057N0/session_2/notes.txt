Binary to Octal
---------------

    To convert 11100101 bin -> oct

    1. Split it into triplets (introducing a leading 0 as its length is 8): 011 100 101

    2. Use the below table:

    000 -> 0
    001 -> 1
    010 -> 2
    011 -> 3
    100 -> 4
    101 -> 5
    110 -> 6
    111 -> 7

    ... to convert 011 100 101

    011 = 3
    100 = 4
    101 = 5

    11100101 === 345 oct


Binary to Decimal
-----------------

    To convert 1110000000 bin -> dec

    1. result = 0
    2. remove the most significant bit (msb)
    3. add it to result
        result = 1
    4. multiply x 2
        result = 2
        110000000
        if no more binary digits left then STOP
        if not, go back to step 2. remove

        result = 2

    1110000000: 0+1   = 1 * 2   = 2
     110000000: 2+1   = 3 * 2   = 6
      10000000: 6+1   = 7 * 2   = 14
       0000000: 14+0  = 14 * 2  = 28
        000000: 28+0  = 28 * 2  = 56
         00000: 56+0  = 56 * 2  = 112
          0000: 112+0 = 112 * 2 = 224
           000: 224+0 = 224 * 2 = 448
            00: 448+0 = 448 * 2 = 896
             0: 896+0 = 896 * 2 = 1792

-----------------

Decimal to Octal
----------------

number  div res     remainder   result
1792 / 8 = 224      0           0
224  / 8 = 28       0           00
28  / 8  = 3        4           004
3   / 8  = 0        3           0043

1792 dec = 0043 oct
